## üöÄ Get Started

Step 1: Install

```bash
$ pnpm add @illa-design/react @emotion/react framer-motion

# or

$ npm i @illa-design/react @emotion/react framer-motion
```

Step 2: Use

```tsx
import { Tag } from '@illa-design/react';

const App = () => (
  <div>
    <Tag>Hello ILLA</Tag>
  </div>
);
```
