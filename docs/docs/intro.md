## üöÄ Get Started

Step 1: Install

```bash
$ yarn add @illa-design/react

# or

$ npm i @illa-design/react
```

Step 2: Use

```tsx
import { Tag } from '@illa-design/react';

const App = () => (
  <div>
    <Tag>Hello ILLA</Tag>
  </div>
);
```
